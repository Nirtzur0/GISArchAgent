No manual downloading or uploading needed!## The Fastest Way to Analyze PlansAn intelligent agentic RAG system for architecture firms to query Israeli planning regulations, approvals, and zoning information from the iPlan system (https://ags.iplan.gov.il/xplan/).

## 🎯 What It Does

Ask natural language questions about:
- 📋 **Israeli Planning Regulations** - TAMA plans, zoning laws, building codes
- 🏛️ **Government Data** - Real-time data from iPlan GIS system
- 🏗️ **Your Projects** - Historical projects, precedents, and approvals
- 📊 **Compliance** - Requirements, procedures, and timelines
- 🖼️ **Visual Plans** - Upload and analyze architectural plans with AI vision

Get intelligent answers powered by:
- 🤖 **LangGraph Agents** - Multi-step reasoning and tool selection
- 🔍 **RAG System** - Retrieval Augmented Generation
- 💾 **Vector Database** - Semantic search across all documents
- 👁️ **Vision AI** - Analyze plan images and ask questions in plain language

## ✨ Features

### Core Capabilities
- **Regulatory Knowledge Base**: Query Israeli planning regulations, zoning laws, and approval processes
- **Automatic Plan Search**: Search iPlan and get **automatic AI vision analysis** of plan images
- **Agentic Architecture**: Multi-agent system with specialized tools for different queries
- **Real-time Data Fetching**: On-demand queries to iPlan without pre-scraping everything
- **Local Project Integration**: Extend to include firm-specific projects and plans
- **Natural Language Queries**: Ask questions in plain language about regulations and plans

### 🔍 Vision-First Plan Analysis (NEW!)
- **Automatic Image Analysis**: When you search for plans, AI automatically analyzes the map images
- **Smart Q&A**: Ask specific questions about visual plans in plain language
- **Manual Upload**: Also upload your own plans for analysis (secondary feature)
- **Token Optimization**: Intelligent caching saves ~85-500 tokens per reuse
- **OCR Support**: Extract Hebrew and English text from images
- **Cost Efficient**: Uses cheapest models (Gemini Flash ~$0.0001 per image)

### 🌐 Beautiful Web Interface
- **🔍 Plan Search**: Search iPlan with automatic vision analysis (PRIMARY FEATURE)
- **💬 Query Assistant**: Interactive Q&A interface for regulations
- **📍 Map Viewer**: Visual TAMA zones and planning schemes on interactive maps
- **📐 Plan Analyzer**: Calculate building rights and compliance automatically
- **🖼️ Image Analyzer**: Manual upload for your own plan documents
- **📊 Database Stats**: Usage analytics and insights

## Architecture

The system uses:
- **LangGraph** for agentic workflow orchestration
- **LangChain** for RAG implementation
- **ChromaDB** for vector storage
- **OpenAI/Google/Anthropic** for LLM capabilities (vision & text)
- **BeautifulSoup/Selenium** for web scraping
- **ArcGIS API** for GIS data extraction
- **Streamlit** for beautiful web interface
- **Pillow** for image processing

## Quick Start

### Installation (5 minutes)

```bash
# 1. Setup
./setup.sh

# 2. Add your API keys
nano .env
# Add:
#   OPENAI_API_KEY=sk-your-key-here
#   GOOGLE_API_KEY=your-gemini-key  # Recommended for vision (cheapest)

# 3. Initialize
python -m src.cli init

# 4. Launch Web App!
streamlit run app.py
```

### Web Interface (Recommended)

```bash
# Start the beautiful web app
streamlit run app.py

# Then navigate to:
# 🔍 Plan Search - Search iPlan with automatic vision analysis
# 📍 Map Viewer - Interactive TAMA zones
# 📐 Plan Analyzer - Building rights calculator
# 🖼️ Image Analyzer - Manual plan upload
```

### Command Line

```bash
# Interactive chat
python -m src.cli query --interactive

# Single query
python -m src.cli query "What are TAMA 35 provisions?"
```

### Python API

```python
from src.main import GISArchAgent

app = GISArchAgent()
response = app.query("What are the main provisions of TAMA 35?")
print(response)
```

## iPlan System Integration

The iPlan system provides:
- **Blue Lines (Kav Kachol)**: Planning schemes across Israel
- **Zoning Layers**: Land use designations
- **TAMA Plans**: National outline plans (TAMA 1, 35, 47, 70)
- **Regional Plans**: District and local planning schemes
- **Section 77-78 Notices**: Planning notifications
